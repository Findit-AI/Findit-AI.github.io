When footage volume grows, retrieval discipline matters more than clever shortcuts.

This workflow is designed for teams that need repeatable speed.

## 1) Keep project structure stable

Use one top-level folder per project or client. Avoid renaming active folders during indexing.

## 2) Query by intent, then refine

Start with scene + action + context.

Example: `interview + walking + city street`

Run broad first, then narrow only after candidate clips appear.

## 3) Preserve context at handoff

When exporting shortlist results to editors:

- keep shared naming conventions
- include why each clip is selected
- save outputs in consistent handoff paths

## 4) Run a weekly failure review

Collect failed or weak queries from the week and update team query patterns.

A short review loop usually improves retrieval quality faster than adding more rules upfront.
